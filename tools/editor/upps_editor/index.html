UPPSエディタについて

                        
このツールはUnified Personality Profile Standard (UPPS)に基づくペルソナプロファイルを編集するためのものです。感情システム、記憶システム、関連性などを統合的に管理できます。

                    

                    
                    

                        
主な機能

                        
    
                            
  • 感情システム: 基本感情のベースライン値と説明を設定
    • 
                            
  • 記憶システム: 異なるタイプの記憶を作成・管理
    • 
                            
  • 関連性ネットワーク: 感情と記憶の間の関連性を設定
    • 
                            
  • 認知能力: WAIS-IVモデルに基づく認知能力の設定
    • 
                            
  • ビジュアライザ: 関連性を視覚的に表示・編集
    • 
                        

                    

                    
                    

                        
ビジュアライザの使い方

                        
    
                            
  • ノードのドラッグ: 配置を調整
    • 
                            
  • ノードのクリック: 選択して詳細表示
    • 
                            
  • Alt+クリック＆ドラッグ: 新しい関連性を作成
    • 
                            
  • ズームコントロール: 右上のボタンで拡大・縮小
    • 
                        

                    

                    
                    

                        
プロファイルの保存と読み込み

                        
画面上部の保存ボタンで現在のペルソナをJSONファイルとして保存できます。また、プレビューパネルからは、JSONまたはYAML形式でエクスポートすることも可能です。30秒ごとに自動保存されるため、ブラウザを閉じても作業を失うことはありません。
